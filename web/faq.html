Frequently Asked Questions

Getting Started

What is MapMyStandards?

MapMyStandards is an AI-powered platform that helps educational institutions map their evidence documents to accreditation standards. It automates the analysis of documents, identifies compliance gaps, and generates comprehensive narratives for accreditation reporting.

How do I get started?

To get started with MapMyStandards:

  • Sign up for an account at platform.mapmystandards.ai/signup
  • Complete the onboarding process to set up your institution profile
  • Select your accreditor (SACSCOC, HLC, MSCHE, etc.)
  • Upload your first evidence documents
  • Review the automated analysis and mappings
Which accreditors are supported?

MapMyStandards currently supports the following accreditors:

  • SACSCOC - Southern Association of Colleges and Schools Commission on Colleges
  • HLC - Higher Learning Commission
  • MSCHE - Middle States Commission on Higher Education
  • NECHE - New England Commission of Higher Education
  • NWCCU - Northwest Commission on Colleges and Universities
  • WASC - Western Association of Schools and Colleges

More accreditors are being added regularly. Contact support if you need a specific accreditor added.

Uploading Documents

What file formats are supported?

MapMyStandards supports the following file formats:

  • PDF - Portable Document Format (recommended)
  • DOCX - Microsoft Word documents
  • DOC - Legacy Word documents
  • TXT - Plain text files

For best results, use PDF format with searchable text (not scanned images).

Is there a file size limit?

Yes, individual files should be under 50MB for optimal processing. If you have larger files:

  • Consider splitting them into smaller documents
  • Compress PDFs using online tools
  • Remove unnecessary images or appendices

You can upload multiple files at once, with a total limit of 500MB per batch.

How long does analysis take?

Analysis time depends on several factors:

  • Small documents (1-10 pages): 30 seconds - 2 minutes
  • Medium documents (10-50 pages): 2-5 minutes
  • Large documents (50+ pages): 5-15 minutes

Processing happens in the background, so you can continue working while documents are analyzed.

Standards & Mapping

How does the mapping algorithm work?

Our AI-powered mapping algorithm:

  • Analyzes document content using natural language processing
  • Identifies key themes, policies, and procedures
  • Compares content against accreditation standards
  • Calculates confidence scores for each mapping
  • Provides detailed explanations for each match

The system learns from user feedback to improve accuracy over time.

Can I manually adjust mappings?

Yes! You have full control over the mappings:

  • Review all automated mappings in the dashboard
  • Add or remove standard associations
  • Adjust confidence levels
  • Add notes and comments for your team
  • Export final mappings for your records
What is the confidence score?

Confidence scores indicate how strongly the AI believes a document matches a standard:

  • High (80-100%): Strong, direct evidence for the standard
  • Medium (60-79%): Relevant but may need additional support
  • Low (40-59%): Tangentially related, review recommended
  • Below 40%: Not shown by default, likely not relevant

Reports & Narratives

What types of reports can I generate?

MapMyStandards offers several report types:

  • Compliance Summary: Overview of standards coverage
  • Gap Analysis: Standards lacking sufficient evidence
  • Evidence Matrix: Detailed mapping of documents to standards
  • Narrative Reports: AI-generated narratives for each standard
  • Executive Summary: High-level overview for leadership

All reports can be exported as PDF, Word, or Excel files.

How are narratives generated?

Our AI generates narratives by:

  • Analyzing all evidence mapped to each standard
  • Extracting key information and themes
  • Writing coherent narratives in academic style
  • Including specific examples from your documents
  • Maintaining your institution's voice and terminology

You can edit and refine all generated narratives before finalizing.

Security & Privacy

Is my data secure?

Yes, we take security seriously:

  • All data is encrypted in transit (TLS 1.3) and at rest (AES-256)
  • Hosted on SOC 2 compliant infrastructure
  • Regular security audits and penetration testing
  • FERPA compliant data handling
  • No sharing of your data with third parties
Who can access my documents?

Only authorized users from your institution can access your documents:

  • You control who has access through user management
  • All access is logged for audit purposes
  • Support staff can only access with your explicit permission
  • Documents are isolated by institution

Troubleshooting

My upload failed. What should I do?

If an upload fails, try these steps:

  • Check your internet connection
  • Ensure the file is under 50MB
  • Verify the file format is supported
  • Try uploading one file at a time
  • Clear your browser cache and try again

If problems persist, contact support with the error message.

The mapping seems incorrect. How can I fix it?

To improve mapping accuracy:

  • Use the feedback buttons to mark incorrect mappings
  • Manually adjust mappings in the review interface
  • Ensure documents have clear, descriptive titles
  • Include context in your documents
  • Contact support for persistent issues
